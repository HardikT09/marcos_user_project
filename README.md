# 🔐 User Management API

A scalable REST API for user authentication and management built with **Node.js**, **Express.js**, **PostgreSQL**, **Sequelize ORM**, and **Redis**. The project demonstrates secure authentication, efficient database operations, and modern backend development practices.

## ✨ Features

* 🔐 JWT Authentication
* 👤 User Registration & Login
* 🔒 Password Hashing with bcrypt
* 🗄️ PostgreSQL Database
* ⚡ Redis Caching
* 📦 RESTful API Design
* 🛡️ Centralized Error Handling
* ✅ CRUD Operations

## 🛠️ Tech Stack

* Node.js
* Express.js
* PostgreSQL
* Sequelize ORM
* Redis
* JWT
* bcrypt

## 🚀 Getting Started

```bash
git clone <repository-url>
cd user-management-api
npm install
npm run dev
```
